CSS
--------------------------
selector {속성 : 값;속성:값;} => 기본형식!!!!
선택자의 의미는 이 속성을 적용 할 대상의 의미한다.

**스타일의 적용법**
- 직접 속성을 추가한다.
- 내부에 style 태그를 추가한다.
- 외부에 .css파일을 작성하여 링크를 건다.
- 우선순위는 직접 속성을 추가한 것이 가장 높다.

**margin 규칙**
- margin : 50 => 상하좌우
- margin : 50 10=> 상하 좌우
- margin : 50 10 5=> 상 좌우 하
- margin : 50 10 5 10=> 상 우 하 좌
